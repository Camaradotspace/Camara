# Anchor

Anchor description.

<br></br>

## Props

Anchor properties

| Property | Description           | Type      | Default | Version |
| -------- | --------------------- | --------- | ------- | ------- |
| href     | Link URL              | `string`  | -       |         |
| asButton | Use anchor as button? | `boolean` | false   |         |
| asNew    | Open link in new tab? | `boolean` | false   |         |

<br></br>

## Usage

### Basic Anchor

```jsx live noInline
import { Anchor, BrowserWindow } from 'camara';

render(
  <BrowserWindow>
    <Anchor href="https://unsplash.com/photos/V_WBdOLOgJU">See photo</Anchor>
  </BrowserWindow>
);
```

### As Button

```jsx live noInline
import { Anchor, BrowserWindow } from 'camara';

render(
  <BrowserWindow>
    <Anchor asButton href="https://unsplash.com/photos/V_WBdOLOgJU">
      See photo
    </Anchor>
  </BrowserWindow>
);
```

### Open In New

```jsx live noInline
import { Anchor, BrowserWindow } from 'camara';

render(
  <BrowserWindow>
    <Anchor asNew href="https://unsplash.com/photos/V_WBdOLOgJU">
      See photo
    </Anchor>
  </BrowserWindow>
);
```
