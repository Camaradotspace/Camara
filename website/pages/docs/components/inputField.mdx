# InputField

InputField description.

<br></br>

## Props

InputField properties

| Property    | Description             | Type                               | Default | Version |
| ----------- | ----------------------- | ---------------------------------- | ------- | ------- |
| helperText  | Input helper text       | `string`                           | -       |         |
| inline      | Should input be inline? | `boolean`                          | false   |         |
| label       | Input label text        | `string`                           | -       |         |
| name        | Input name              | `string`                           | -       |         |
| placeholder | Input placeholder text  | `string`                           | -       |         |
| required    | Is input required?      | `boolean`                          | false   |         |
| type        | What input type is it?  | `number` `text` `email` `password` | `text`  |         |

<br></br>

## Usage

### Basic Input

```jsx live noInline
import { InputField, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <InputField placeholder="Type something here" />
  </CamaraTheme>
);
```

### Block

```jsx live noInline
import { Box, InputField, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Box>
      <InputField label="First Name" />
      <InputField label="Last Name" />
    </Box>
  </CamaraTheme>
);
```

### Inline

```jsx live noInline
import { Flex, InputField, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Flex>
      <InputField inline label="First Name" />
      <InputField inline label="Last Name" />
    </Flex>
  </CamaraTheme>
);
```

### Helper Text

```jsx live noInline
import { Flex, InputField, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Flex>
      <InputField
        label="Email Address"
        type="email"
        helperText="example@gmail.com"
      />
      <InputField
        label="Password"
        type="password"
        helperText="must contain special characters"
      />
    </Flex>
  </CamaraTheme>
);
```

### Required

```jsx live noInline
import { Flex, InputField, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Flex>
      <InputField required label="First Name" />
      <InputField required label="Last Name" />
    </Flex>
  </CamaraTheme>
);
```

### Hidden Label

```jsx live noInline
import { Flex, InputField, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Flex>
      <InputField required placeholder="First Name" />
      <InputField required placeholder="Last Name" />

```

### Disabled

```jsx live noInline
import { Flex, InputField, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Flex>
      <InputField disabled required placeholder="First Name" />
      <InputField disabled required placeholder="Last Name" />

```
