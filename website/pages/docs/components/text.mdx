# Text

Text description.

<br></br>

## Props

Text properties

| Property  | Description                    | Type                                               | Default  | Version |
| --------- | ------------------------------ | -------------------------------------------------- | -------- | ------- |
| align     | Set text alignment             | `left` `right` `center` `justify`                  | `left`   |         |
| color     | Text color                     | `string`                                           | -        |         |
| inline    | Should text be inline?         | `boolean`                                          | false    |         |
| italic    | Should text be in italics?     | `boolean`                                          | false    |         |
| size      | Set size of text               | `small` `medium` `large`                           | `large`  |         |
| strike    | Should text be strike-through? | `boolean`                                          | false    |         |
| style     | Set style of text              | `CSSProperties`                                    | -        |         |
| underline | Should text be underlined?     | `boolean`                                          | false    |         |
| variant   | What type of text is it?       | `body` `caption` `sub-heading` `overline` `strong` | `body`   |         |
| weight    | Set font-weight of text        | `bold` `normal`                                    | `normal` |         |

<br></br>

## Usage

### Basic Text

```jsx live noInline
import { Text } from 'camara';

render(
  <>
    <Text>This is a text component</Text>
  </>
);
```

### Variant

```jsx live noInline
import { Box, Text } from 'camara';

render(
  <>
    <Box>
      <Text variant='body'>I am a body text</Text>
      <Text variant='sub-heading'>I am a sub heading</Text>
      <Text variant='caption'>I am a caption</Text>
      <Text variant='overline'>I am an overline</Text>
      <Text variant='strong'>I am a strong text</Text>
    </Box>

    <Box>
      <Text variant='body'>
        私の夢の仕事は、Microsoft❤でソフトウェアエンジニアとして働くことです。
      </Text>
      <Text variant='sub-heading'>
        私の夢の仕事は、Microsoft❤でソフトウェアエンジニアとして働くことです。
      </Text>
      <Text variant='caption'>
        私の夢の仕事は、Microsoft❤でソフトウェアエンジニアとして働くことです。
      </Text>
      <Text variant='overline'>
        私の夢の仕事は、Microsoft❤でソフトウェアエンジニアとして働くことです。
      </Text>
      <Text variant='strong'>
        私の夢の仕事は、Microsoft❤でソフトウェアエンジニアとして働くことです。
      </Text>
    </Box>
  </>
);
```
