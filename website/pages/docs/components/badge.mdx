# Badge

Badge description here.

<br></br>

## Props

Badge properties

| Property        | Description                                  | Type                     | Default  | Version |
| --------------- | -------------------------------------------- | ------------------------ | -------- | ------- |
| backgroundColor | Background color of the badge                | `string`                 | -        |         |
| color           | Color of the badge text                      | `string`                 | -        |         |
| text            | Badge text                                   | `ReactNode`              | -        |         |
| shape           | Shape of the badge                           | `circle` `square` `pill` | `square` |         |
| size            | Size of the badge                            | `large` `medium` `small` | `small`  |         |
| position        | Position of badge relative to parent element | `middle` `top`           | -        |         |

<br></br>

## Usage

### Size

```jsx live noInline
import { Badge, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Badge size="large" text="Preview" />
    <Badge size="medium" text="Preview" />
    <Badge size="small" text="Preview" />
  </CamaraTheme>
);
```

### Shape

```jsx live noInline
import { Badge, Box, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Box>
      <Badge shape="square" size="large" text="Preview" />
      <Badge shape="square" size="medium" text="Preview" />
      <Badge shape="square" size="small" text="Preview" />
    </Box>
    <Box>
      <Badge shape="pill" size="large" text="Preview" />
      <Badge shape="pill" size="medium" text="Preview" />
      <Badge shape="pill" size="small" text="Preview" />
    </Box>
    <Box>
      <Badge shape="circle" size="large" text="M" />
      <Badge shape="circle" size="medium" text="M" />
      <Badge shape="circle" size="small" text="M" />
    </Box>
  </CamaraTheme>
);
```
