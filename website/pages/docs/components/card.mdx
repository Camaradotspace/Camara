# Card

Card description.

<br></br>

## Props

Card properties

| Property | Description                                            | Type      | Default | Version |
| -------- | ------------------------------------------------------ | --------- | ------- | ------- |
| bordered | Should card have border around it or be entirely flat? | `boolean` | false   |         |
| elevate  | Should the card elevate on the z-index?                | `boolean` | false   |         |
| height   | Height of the card                                     | `string`  | -       |         |
| width    | Width of the card                                      | `string`  | -       |         |

<br></br>

## Usage

### Elevate

```jsx live noInline
import { Button, Card, Image, Text, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Card elevate>
      <Image
        imageUrl="https://source.unsplash.com/V_WBdOLOgJU/400x300"
        imageAlt="woman in orange knit cap and blue jacket"
      />
      <Text>John Doe</Text>
      <Button block pill size="large">
        Follow
      </Button>
    </Card>
  </CamaraTheme>
);
```

### Bordered

```jsx live noInline
import { Box, Button, Card, Image, Text, Spacer, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Box>
      <Box>
        <Card>
          <Image
            imageUrl="https://source.unsplash.com/V_WBdOLOgJU/400x300"
            imageAlt="woman in orange knit cap and blue jacket"
          />
          <Text>John Doe</Text>
          <Button block pill size="large">
            Follow
          </Button>
        </Card>
      </Box>

      <Spacer top>
        <Box>
          <Card bordered={false}>
            <Image
              imageUrl="https://source.unsplash.com/V_WBdOLOgJU/400x300"
              imageAlt="woman in orange knit cap and blue jacket"
            />
            <Text>John Doe</Text>
            <Button block pill size="large">
              Follow
            </Button>
          </Card>
        </Box>
      </Spacer>
    </Box>
  </CamaraTheme>
);
```
