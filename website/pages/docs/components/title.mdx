# Title

Title description.

<br></br>

## Props

Title properties

| Property | Description                                             | Type                              | Default | Version |
| -------- | ------------------------------------------------------- | --------------------------------- | ------- | ------- |
| align    | Set Title alignment                                     | `left` `right` `center` `justify` | `left`  |         |
| color    | Title color                                             | `string`                          | -       |         |
| level    | What heading level should title be? h1, h2, h3, h4, etc | `number`                          | 1       |         |
| thin     | Control thickness of the title                          | `boolean`                         | false   |         |

<br></br>
