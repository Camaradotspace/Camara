# Image

Image description.

<br></br>

## Props

Image properties

| Property | Description                                                             | Type                                         | Default | Version |
| -------- | ----------------------------------------------------------------------- | -------------------------------------------- | ------- | ------- |
| imageUrl | Link to image source                                                    | `string`                                     | -       |         |
| imageAlt | Description of the image                                                | `string`                                     | -       |         |
| loading  | Control if loading the image should be deferred when its off the screen | `eager` `lazy`                               | -       |         |
| fit      | Object-fit property of the image                                        | `fill` `contain` `cover` `none` `scale-down` | `cover` |         |

<br></br>

## Usage

### Basic Image

```jsx live noInline
import { Image, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Image
      imageUrl="https://source.unsplash.com/V_WBdOLOgJU/600x500"
      imageAlt="woman in orange knit cap and blue jacket"
    />
  </CamaraTheme>
);
```

### Loading

```jsx live noInline
import { Flex, Image, CamaraTheme } from 'camara';

render(
  <CamaraTheme>
    <Flex justify="space-between">
      <Image
        width="49%"
        imageUrl="https://source.unsplash.com/V_WBdOLOgJU/400x300"
        imageAlt="woman in orange knit cap and blue jacket"
        loading="eager"
      />

      <Image
        width="49%"
        imageUrl="https://source.unsplash.com/V_WBdOLOgJU/400x300"
        imageAlt="woman in orange knit cap and blue jacket"
        loading="lazy"
      />
    </Flex>
  </CamaraTheme>
);
```
