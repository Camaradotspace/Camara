# Button

## Examples

```jsx live noInline
import { Button } from 'camara';

render(<Button>A simple button</Button>);
```

### Primary

```jsx live noInline
import { Button } from 'camara';

render(<Button primary={true}>A simple button</Button>);
```

### Background Color

```jsx live noInline
import { Button } from 'camara';

render(
  <Button primary={true} backgroundColor='pink' color='red'>
    A simple button
  </Button>
);
```

### Rounded

```jsx live noInline
import { Button } from 'camara';

render(
  <Button primary={true} backgroundColor='pink' rounded={true} color='red'>
    A simple button
  </Button>
);
```

## API

Button properties

| Property | Description                          | Type      | Default | Version |
| -------- | ------------------------------------ | --------- | ------- | ------- |
| loading  | Set the loading status of the button | `boolean` | false   |         |
